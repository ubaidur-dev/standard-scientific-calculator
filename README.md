# Standard Scientific Calculator 
This is a clean, high-quality scientific calculator I built with a focus on tactile UI and mathematical precision. Instead of making just another flat tool, I focused on creating a "3D Glassmorphism" look that gives the buttons a realistic feel while keeping the underlying logic fast and modular.

## Key Features
* **Scientific Operations:** I integrated advanced functions like Square Root, Powers, and PI to handle more than just basic arithmetic.
* **3D Tactile Design:** I used custom CSS depth and shadows to give the buttons a realistic, physical look. It’s not just a flat UI; it feels like a real device.
* **Smart Input Logic:** I wrote custom JavaScript to handle common user errors, like preventing double operators or division by zero, so the app doesn't crash.
* **Pure CSS Grid:** To keep the layout pixel-perfect, I used CSS Grid. This ensures every button is perfectly aligned and stays that way across mobile and desktop.
* **Zero Dependencies:** The entire app loads instantly because I didn't use any heavy libraries or frameworks. It’s pure, raw performance.

## Why I Built This
My goal was to show how small details like proper depth and tactile effects can make a simple utility feel like a premium application. I also wanted to show how a complex scientific tool can be kept organized by separating the logic and styling into clean, readable modules.

## Tech Stack
* **HTML5:** For a clean, semantic structure that keeps the buttons organized and accessible.
* **CSS3:** For custom 3D styling, smooth transitions, and the responsive Grid layout.
* **Vanilla JavaScript:** For fast, lightweight math operations without the need for external libraries.

## How to Use
* Clone or download this repository.
* Open the **index.html** file in any modern web browser.
* That's it! No installation or setup required.
