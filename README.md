# standard-scientific-calculator
Simple scientific calculator with a clean 3D look. Built using basic JS and CSS grid to keep it fast and modular.
A clean and simple scientific calculator built for a smooth experience. I focused on keeping the code light and the design modern so it works great on any screen.

Tech & Tools:
Logic: Pure JavaScript (No heavy frameworks or libraries).
Layout: Built with CSS Grid to keep the buttons perfectly aligned.
Design: Used 3D glassmorphism to give it a realistic, tactile feel.
Responsive: Optimized for mobile, tablet, and desktop views.

Key Features:
Scientific Operations: Handles complex stuff like Square Root, Powers, and PI.
Smart Logic: Built-in error handling to prevent common math crashes.
Fast & Light: Zero dependencies, meaning it loads instantly in any browser.
Clean UI: A minimal and professional look that's easy on the eyes.

Development Note:
The goal was to create a modular component that is easy to read and maintain. I wanted to show how a simple tool can be made professional by focusing on clean code and solid UI design.
